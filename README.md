# 📜 陰債計算系統 

📜 **輸入生辰八字，自動計算陰債總額，適合宗教相關網站使用並搭配提供運勢建議！**

##  **使用方式**
1. **選擇出生日期 & 時辰** 
2. **點擊 `送出` 按鈕，查看陰債結果**  
3. **如不知時辰，可勾選 `不知道生辰`，計算範圍**  
4. **可點擊 `複製結果`，分享到 LINE / Facebook**
